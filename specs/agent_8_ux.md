## 4. Accessibility & Low-Bandwidth Considerations

### Accessibility

| Feature | Implementation |
|---------|----------------|
| **Color Contrast** | WCAG AA minimum (4.5:1 text, 3:1 UI) |
| **Touch Targets** | Minimum 44x44px for all tappable elements |
| **Font Sizes** | Base 16px, minimum 14px, scalable via system |
| **Screen Reader** | ARIA labels on all interactive elements |
| **Error States** | Color + icon + text (never color alone) |
| **Reduced Motion** | Respect `prefers-reduced-motion` |
| **Focus States** | Visible focus rings on keyboard navigation |

### Amharic Language Support

```
CONSIDERATIONS:
- Geez script support (Unicode)
- Right-to-left NOT required (Amharic is LTR)
- Longer text strings (plan for 1.5x English length)
- Font: Noto Sans Ethiopic or system default
- Number formatting: Use Ethiopian numerals option
- Date formatting: Both Gregorian and Ethiopian calendar
```

### Low-Bandwidth Optimizations

| Optimization | Target |
|--------------|--------|
| **Initial Load** | < 500KB (gzipped) |
| **Image Lazy Loading** | Below-fold images deferred |
| **Image Compression** | WebP with JPEG fallback |
| **Avatar Thumbnails** | 50x50px @ 5KB max |
| **Player Photos** | 100x100px @ 15KB max |
| **API Responses** | Paginated (20 items default) |
| **Caching** | Service Worker for offline core UI |
| **Data Saver Mode** | Option to disable images |

### Offline Support

```
CACHED (Available Offline):
- Home screen (last state)
- My Team (current squad)
- Recent trivia questions
- Player database (local SQLite)

NOT CACHED (Requires Connection):
- Real-time leaderboards
- Live scores
- Payment flows
- QR scanning
```

### Network Error Handling

```
TIMEOUT (>10 seconds):
"Slow connection. [Retry] or [Continue Offline]"

NO CONNECTION:
"You're offline. Some features are unavailable.
Your changes will sync when you reconnect."

PARTIAL DATA:
[Show cached data]
"Last updated 2 hours ago. [Refresh]"
```

## Key Decisions (Agent 8)

1. **Modal-first for check-in**: Don't let users miss daily reward
2. **5-step onboarding max**: Minimize drop-off, collect minimal data
3. **Explicit compliance language**: Disclaimer on every contest screen
4. **Amharic as first-class**: Full translation, not afterthought
5. **Offline-first mentality**: Core features work without connection

## Assumptions

- Most users on Android with mid-range devices
- 3G/4G connectivity, not always reliable
- Users familiar with mobile payment apps (Telebirr)
